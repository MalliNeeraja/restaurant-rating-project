# Restaurant Rating Prediction Project

This project is part of my **Data Science Internship at Cognifyz Technologies**.  
It predicts restaurant ratings using data science and machine learning techniques.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Visualization

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

## How to Run
1. Open the notebook: restaurant_rating_project.ipynb  
2. Run all cells in Jupyter Notebook or Google Colab  
3. Make sure the dataset file is in the same folder
